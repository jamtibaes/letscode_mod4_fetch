# letscode_mod4_fetch

https://jamtibaes.github.io/letscode_mod4_fetch/


## O que é?

Trabalho de conclusão do modulo 4 da Let's Code Santander onde foi proposto fazer um fetch em uma API que retorna informações da tabela FIPE 


## Funcionamento
Selecione na caixa acima o tipo de veiculo (carro, moto ou caminhão) e retornara uma lista na caixa abaixo com as marcas relacionados ao tipo do veículo. Será apresentado os dados da escolha.
